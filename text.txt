如何在Vue2.0中集成SVG图标 ？

通过vue-svg-icon

1. 安装：npm install vue-svg-icon --save-dev

2. 在项目的src路径下新建svg路径，将下载的svg资源放入svg目录下，如right.svg

3. main.js中
import Vue from "vue" ;
import Icon from 'vue-svg-icon/Icon.vue';
Vue.component('icon', Icon);

4. 在组件中，直接使用
<icon name="right" :scale="4"></icon>
